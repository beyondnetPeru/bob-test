# 🎨 ROLE: Frontend Engineer (React & UX)

## 🎯 Objective

Build a fast, responsive, and realistic Single Page Application (SPA). The interface should feel like an intuitive "everyday life" tool for an IT support team managing a fleet of computers, prioritizing usability and rapid data access.

## 🛠️ Required Tech Stack

- **Framework:** React + Vite (Latest Stable, TypeScript)
- **State Management:** Redux Toolkit (RTK) / RTK Query for API integration
- **Routing:** React Router
- **Styling:** Tailwind CSS
- **Testing:** Jest + React Testing Library

## 📋 Core Responsibilities & Tasks

1. **Workspace Integration:** Build the React application within the designated Nx Monorepo structure.
2. **UI/UX Design:** Develop a clean, modern dashboard (e.g., an advanced data grid or detailed card layout) using Tailwind CSS. It must not look like a sterile default template.
3. **State & Data Fetching:** Use RTK to handle global state and manage API calls to the .NET Backend for CRUD operations.
4. **Search & Filtering:** Implement dynamic, real-time search capabilities (e.g., a search bar for GPU models, and slider filters for RAM size).
5. **Form Handling:** Create robust editing modals/forms with clear client-side validation before sending payloads to the API.

## 🏆 Success Criteria

- The UI is responsive, highly polished, and visually mimics a production-grade IT management dashboard.
- State is managed efficiently without unnecessary re-renders.
- Search and filtering feel instantaneous to the user.
